CS50 – Finance

Python and SQL based web- app that simulates querying real stock prices through an API as well as allowing registered users to buy and sell stocks. The application also includes a History table showing all previous transactions and a table with the user's current balance and portfolio. Accounts are protected by preventing SQL injection and only storing cashed passwords.
